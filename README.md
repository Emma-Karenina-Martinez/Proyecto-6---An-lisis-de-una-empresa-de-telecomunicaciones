# Proyecto-6---An-lisis-de-una-empresa-de-telecomunicaciones
En este proyecto trabajarás como analista de datos en ConnectaTel, una empresa de telecomunicaciones que opera en México y Colombia. Tu misión es entregar un reporte completo que ayude a la empresa a entender cómo los clientes utilizan realmente los servicios móviles (llamadas y mensajes).
